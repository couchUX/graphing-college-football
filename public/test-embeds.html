CFB Chart Embed Tester

How to use:
1. Copy embed code from your local dashboard (http://localhost:5173)
2. Paste it in the textarea below
3. Click "Render Embed" to see how it looks
4. Use Cmd/Ctrl + R to hard reload and test consistency
5. Mimics 600px article width (like WordPress)
💡 Tip: After rendering, try hard reloading (Cmd+Shift+R / Ctrl+Shift+R) to test if labels stay in correct position
